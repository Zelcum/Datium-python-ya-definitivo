Guía de Importación

Formato de Archivo

Aceptamos archivos de tipo JSON y CSV. El sistema mapeará automáticamente las columnas basándose en el nombre de los campos.

Reglas Técnicas:

  • Nombres de Columnas: Deben coincidir con los nombres de los campos de la tabla. No distinguen mayúsculas/minúsculas.
  • Relaciones: Puedes usar el ID numérico del registro relacionado o el valor del campo principal (ej: nombre de cliente).
  • Fechas: Use el formato estándar YYYY-MM-DD.
  • Booleanos: Use true/false, 1/0 o Sí/No.

Ejemplo CSV:

Nombre,Edad,Activo,Ciudad_Relacion\n"Juan Perez",30,Sí,1\n"Maria Lopez",25,No,2
